# Frontend часть приложения


xs: 4px (1) — микро-отступы (между иконкой и текстом).
sm: 8px (2) — малые отступы (внутри элементов).
md: 16px (4) — стандартный отступ (внутри карточек, кнопок).
lg: 24px (6) — между блоками в одной секции.
xl: 32px (8) — между логическими разделами.
2xl: 48px (12) — для Hero-секции.
3xl: 64px (16) — большие отступы между секциями.
